  <h1>Steps to do: </h1>
      <ol type = "1">
         <li>Clone the repo</li>
         <li>Install MySQL DB</li>
         <li>Run DB</li>

         the property user to your username and password for your password(MySQL of course :)
         </li>
         <li>Build and run the app by writing dotnet run</li>
      </ol>
